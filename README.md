# Portfolio Audiovisual — Juan Javier Bordería Pi

Repositorio oficial del portfolio web de **Juan Javier Bordería Pi**, Periodista, Comunicador Audiovisual y Editor.

El sitio está diseñado para presentar de manera clara, estructurada y sin intermediarios una selección de proyectos en los que he participado con diferentes roles técnicos y creativos:
- **Producción Propia**: Proyectos creados íntegramente (cámara, guion, montaje y producción).
- **Edición y Postproducción**: Piezas multicámara y corporativas montadas en Adobe Premiere Pro.
- **Locución**: Trabajos de voz en off y narración para vídeos informativos e institucionales.
- **Redes Sociales (RRSS)**: Contenidos nativos optimizados para plataformas verticales y horizontales (Instagram y YouTube).

## Arquitectura y Tecnologías
Este proyecto web está desarrollado bajo una filosofía de simplicidad, alto rendimiento y compatibilidad universal sin frameworks pesados:
- **HTML5 Semántico**: Estructuración clara de contenidos enfocada en la accesibilidad.
- **CSS3 Nativo**: Sistema de diseño a medida implementando variables (CSS custom properties), modo oscuro optimizado, tipografías fluidas y diseño adaptable (responsive design) a cualquier dispositivo.
- **JavaScript (ES6)**: Gestión ligera del DOM, sistema nativo de filtrado por categorías y visualización inmersiva tipo *lightbox* para la reproducción directa de contenidos multimedia.
